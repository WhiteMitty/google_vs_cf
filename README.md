# google_vs_cf
Choose between Google and CF DNS and lock them.

## 安装

### curl

```bash
curl -fsSL -o google_vs_cf.sh https://raw.githubusercontent.com/WhiteMitty/google_vs_cf/main/google_vs_cf.sh && bash google_vs_cf.sh
```

### wget

```bash
wget -qO google_vs_cf.sh https://raw.githubusercontent.com/WhiteMitty/xray-manager/main/google_vs_cf.sh && bash google_vs_cf.sh
```
